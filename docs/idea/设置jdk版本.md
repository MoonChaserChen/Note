# 设置jdk版本
https://www.jetbrains.com/help/idea/troubleshooting-common-maven-issues.html#maven_compiler_problem

## JavaCompiler和LanguageLevel
```xml
<build>
    <plugins>
        <plugin>
            <groupId>org.apache.maven.plugins</groupId>
            <artifactId>maven-compiler-plugin</artifactId>
            <version>3.7.0</version>
            <configuration>
                <source>1.8</source>
                <target>1.8</target>
                <encoding>UTF-8</encoding>
            </configuration>
        </plugin>
    </plugins>
</build>
```
